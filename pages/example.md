# 환영합니다!

안녕하세요! **GitHub Pages**로 만든 블로그에 오신 것을 환영합니다.

이 게시글은 마크다운 문법과 블로그 기능을 테스트하기 위한 샘플 게시글입니다.

## 마크다운 문법 테스트

### 텍스트 스타일

- **굵은 글씨**
- *기울임꼴*
- ~~취소선~~
- `인라인 코드`

### 링크와 이미지

[GitHub](https://github.com)에서 더 많은 정보를 확인하세요.

### 인용문

> 좋은 코드는 그 자체로 최고의 문서이다.
> — 스티브 맥코넬

### 코드 블록

JavaScript 예제:

```javascript
function greeting(name) {
  console.log(`안녕하세요, ${name}님!`);
}

greeting('방문자');
```

Python 예제:

```python
def greeting(name):
    print(f"안녕하세요, {name}님!")

greeting("방문자")
```

## 다음 단계

새 게시글을 작성하려면 `pages/` 폴더에 `.md` 파일을 추가하세요.

파일 상단에 Front Matter를 포함하는 것을 잊지 마세요:

```yaml
---
title: '게시글 제목'
date: 2025-01-01
tags: ["태그1", "태그2"]
category: '카테고리명'
description: '게시글 설명'
---
```

즐거운 블로깅 되세요! 🎉
